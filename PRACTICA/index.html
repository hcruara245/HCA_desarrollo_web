Resumen de todas las etiquetas de HTML (ejemplo de tabla 2)
Etiqueta Descripción
<h1> a <h6> Encabezados de diferentes niveles
<p> Párrafo de texto
<a> Enlace a otra página web
<img> Inserta una imagen
<ul>, <ol>, <li> Listas no ordenadas y ordenadas
<video> Reproducción de vídeo
<table>, <tr>, <th>, <td> Tabla de datos
<form>, <input>, <select>, <textarea> Formulario de entrada de datos
<canvas> Área de dibujo gráfico
<audio> Reproducción de audio
