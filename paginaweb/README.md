Autoescuela Máxima - Página web simple

Descripción
-----------
Pequeña página informativa para la autoescuela "Autoescuela Máxima". Incluye HTML y CSS con un diseño moderno, responsive y animaciones ligeras.

Archivos
--------
- `index.html` - Página principal.
- `style.css` - Estilos, layout y animaciones.

Cómo probar
-----------
1. Abrir `index.html` en tu navegador (doble clic o arrastrar al navegador).
2. Para desarrollo, puedes servirla con un servidor estático (opcional):

   - Si tienes Python 3 instalado:

```powershell
cd ruta\a\paginaweb
python -m http.server 8000
# Abrir http://localhost:8000
```

Notas
-----
- El formulario de contacto sólo muestra una alerta (no envía datos).
- Si quieres que añada iconos, más animaciones o integraciones, dime qué prefieres.
